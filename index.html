Screenshot Guidelines

  • If you're about to create an article for a new or undocumented feature within a studio, make sure to create an image folder for it within the main image folder.
  • Example: This is the path to an image for a Zoe document within the Advisor Studio > Zoe folder:
    \partner-documentation\docs\advisor-studio\zoe\img\zoe_editor.png
  • Find your image on the list of files, right-click and select "Copy Relative Path". Paste the relative path inside brackets.
  • Use the "BrowserWindow" component to add shadow borders to your images (you can replace the link below - url="https://orca-advisor.zoovu.com") with the link to any studio:
<BrowserWindow url="https://orca-advisor.zoovu.com">

![Screenshot description](img/zoe_editor.png)

</BrowserWindow>

For the BrowserWindow component to work, please make sure to add this line to your article right below the metadata:
import BrowserWindow from "@site/src/components/BrowserWindow";

Add notes/warnings

Docusaurus allows you to add admonitions to your articles:

:::note

This is a note.

:::


:::tip

This is a helpful tip.

:::


:::info

This is an info box.

:::


:::warning

This is a warning. Try to use it only where absolutely necessary - warnings make people anxious.

:::


:::danger

This is the scariest type of warning.

:::

Restrict access to a document

Sometimes you may want to make a doc accessible only to other Zoovu employees or a specific Client. For example, when ZOE was first developed we only wanted selected Clients to know about it.

  • If it's for internal use only, you can use this repository for such content: https://gitlab.zoovu.io/technical-documentation/internal-documentation
  • If it's a doc for specific Clients, you can use the unlisted feature provided by Docusaurus (the unlisted: true front matter option) to hide certain pages from production build. They will be accessible via direct links. Unlisted content will also display a banner so that you don't forget to turn it off once your content is ready for publishing.
